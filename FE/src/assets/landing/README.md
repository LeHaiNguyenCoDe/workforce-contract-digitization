# Assets

## Images

Chứa hình ảnh cho Landing page:

- `logo.svg` - Logo
- `hero/` - Hình ảnh hero section
- `products/` - Hình ảnh sản phẩm placeholder
- `icons/` - Icons tùy chỉnh

## SCSS

Chứa styles cho Landing page:

- `index.scss` - Entry point
- `_variables.scss` - Variables (colors, spacing...)
- `components/` - Component styles
- `pages/` - Page-specific styles
